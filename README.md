A simple Notes App built with Flutter using sqflite for local database storage. This project demonstrates full CRUD operations (Create, Read, Update, Delete) with a clean UI.

📱 Features

✅ Add new notes
✅ View all notes
✅ Edit existing notes
✅ Delete notes
✅ Data persists locally (offline support)
✅ Clean and simple UI

📦 Dependencies
Add these to your pubspec.yaml:

dependencies:
  flutter:
    sdk: flutter
  sqflite: ^2.3.0
  path: ^1.8.3
